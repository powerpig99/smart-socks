Abstract

Smart Socks is a wearable system that recognizes human activities in real-time using piezoresistive fabric sensors embedded in footwear. The system captures pressure distribution patterns at 50 Hz and classifies 11 different activities including walking, stairs, sitting, and standing postures with 94.2% accuracy using a Random Forest classifier.

Introduction

Wearable technology for activity recognition has significant applications in healthcare, sports training, and elderly monitoring. Traditional systems rely on inertial measurement units (IMUs) which can be intrusive and power-hungry.

Our approach uses piezoresistive fabric sensors that are comfortable, washable, and require minimal power while providing rich pressure distribution data for accurate activity classification.

System Architecture

  • 10 piezoresistive fabric sensors (5 per sock)
  • ESP32S3 XIAO microcontroller (240 MHz)
  • 12-bit ADC at 50 Hz sampling rate
  • WiFi + BLE wireless connectivity
  • Edge computing with real-time classification

Sensor Placement

Pressure sensors are strategically placed at key anatomical locations:

  • Heel: Initial contact detection
  • Arch: Mid-stance support analysis
  • Metatarsals: Push-off force measurement
  • Toe: Balance and propulsion
Sensor Placement Diagram

Methodology

Feature Extraction: Statistical and frequency-domain features (185 per window)

Classification: Random Forest with cross-subject validation

Windowing: 1-second windows with 50% overlap

Validation: Leave-one-subject-out cross-validation to ensure generalization

Applications

  • Elderly fall risk assessment
  • Rehabilitation progress monitoring
  • Sports performance analysis
  • Gait disorder diagnosis
  • Activity tracking for fitness
