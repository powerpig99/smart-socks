## Aalto Library Access Instructions

### Option 1: Aalto VPN
1. Install Cisco AnyConnect VPN client
2. Connect to `vpn.aalto.fi`
3. Log in with Aalto credentials
4. Access papers through publisher websites

### Option 2: Aalto-Primo
1. Go to: https://aalto-primo.hosted.exlibrisgroup.com/
2. Search for paper title
3. Click "Full text available" link
4. Log in with Aalto credentials if prompted

### Option 3: Direct Proxy
Replace publisher URL with Aalto proxy:
- Original: `https://www.sciencedirect.com/...`
- With proxy: `https://www.sciencedirect.com.ezproxy.aalto.fi/...`
